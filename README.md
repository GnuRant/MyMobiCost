MyMobiCost
==========
Calcola il costo della vita nella provincia di Belluno

# Backend

###Installazione pip
```bash
  curl https://raw.github.com/pypa/pip/master/contrib/get-pip.py > get-pip.py
```
```bash
  python  get-pip.py
```
###Installazione flask e pg8000
```bash
  pip install Flask
  pip install pg8000
```
###Avviare il server 
```
  python run.py
```
