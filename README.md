MyMobiCost
==========
Calcola il costo della vita nella provincia di Belluno

# Backend

###Installazione pip
```
  curl https://raw.github.com/pypa/pip/master/contrib/get-pip.py > get-pip.py
```
```
  python  get-pip.py
```
###Installazione flask
```
  pip install Flask
```
###Avviare il server 
```
  python run.py
```
####Pagina test
```
http://0.0.0.0:5000/status
```
